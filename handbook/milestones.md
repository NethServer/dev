# Milestones

Utilizzo delle milestone per l'organizzazione dei progetti.
Le issue assegnate a una milestone avranno il tipo "draft".
Ogni sviluppatore può assegnarsi le issue liberamente per rendere visibile il proprio lavoro imminente.
Gli assegnatari delle issue vengono rimossi quando la issue passa in QA.


Ogni rilascio di una milestone richiede un annuncio.
